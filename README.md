# SofiaTalks Website

Official website for **SofiaTalks**, one of Bulgaria’s leading youth career and entrepreneurship events.

## About the Project

The Sofia Talks website is created to support the online presence of the event and communicate its identity, purpose, and opportunities to visitors, participants, and partners.

It serves as a digital platform for showcasing the event and maintaining a professional visual presence.

## My Role

- Co-designed the website
- Worked on the visual direction and structure
- Contributed to the presentation of the event online
